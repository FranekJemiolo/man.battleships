# man.battleships.ui
Front end for Battleships game

## Files
### index.html
HTML for the front end

### battleships.js
Main vue controller for the UI

### game.json
Is a placeholder json for local testing without the corresponding python backend

### teams.json
Is a placeholder json for local testing without the corresponding python backend

## Libraries used
* VueJS
* Axios
* Bootstrap
  * Jquery
  * Popper

These are all loaded via CDN
